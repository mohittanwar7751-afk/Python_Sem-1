Daily Calorie Tracker CLI

This project is a Python-based console application that allows users to record their daily meals and calories. It helps track total calories, average calories per meal, and compares the total with a personal daily limit. Users can also save their session report to a text file.

⸻

Features
	•	Enter meal names and calorie values
	•	Stores data using lists
	•	Calculates total and average calorie intake
	•	Shows warning if daily limit is exceeded
	•	Displays a clean, formatted summary
	•	Optional feature to save the session log

⸻

Project Contents
	•	tracker.py — Main program
	•	calorie_log.txt — Optional saved session log
	•	README.md — Project description
